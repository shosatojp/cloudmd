# CloudMarkdown
https://cloudmd.monoid.app/

## Features
* Compile Markdown to PDF (md -> tex -> pdf)
* Compile raw Tex

## Default Tex Packages
* booktabs
* longtable
* framed
* amsmath
* caption
* hyperref
* graphicx

## Common Use Case
1. Maintain your report directories with git.
1. Use CloudMd only when you compile.




## How to Build (for Developers)
```sh
make
```
